简单介绍一下安装吧(大神略过)
游戏安装大的步骤分为三步
第一步：
1、安装mysql数据库(tidb也行)，这个我就不教了，也很简单。如果有什么问题的话我有一个文章专门介绍怎么安装数据库的
2、把项目中config目录下的maple_20201013.sql 文件还原到安装好的数据库中，这就是整个项目的数据库了，里面有现成的数据，有些表我还没摸透是什么意思，大家一起来完善完善
3、修改一下config下面的db.properties 文件中的数据库连接信息就行了

第二步：编译和打包运行服务端
这部分现在也相当简单了，我把代码放在了git上，大家可以本地直接拉下来，以后有更新也可以直接git pull 拉去最新的
所有的操作都是脚本化的，双击一下就行了，具体也看下我发的那个文章吧，也很简单

第三步：运行游戏
下载个客户端，双击登录器就进入游戏了
